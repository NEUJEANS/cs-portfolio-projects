Two-phase commit incident-response dashboard

Blocked-case triage at a glance

This static dashboard strips the portfolio bundle down to the incidents that actually block after PREPARE. It groups them by the response story that matters most during triage: waiting for coordinator recovery, following peer-visible COMMIT evidence, or proving a safe ABORT from a peer that never reached PREPARED.
